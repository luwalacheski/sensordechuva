Componentes do Varal Automático com Sensor de Chuva

Sensor de Chuva FC-37

Este é o sensor FC-37, responsável por detectar gotas de chuva. Ele envia um sinal elétrico para o Arduino quando identifica umidade na placa. É o componente principal para ativar o recolhimento automático do varal.

Função: Detectar chuva e acionar o sistema automático.

Preço: R$ 29,90

Esquema com Arduino Nano

Este circuito mostra como o sensor FC-37 é conectado ao Arduino Nano e ao display LCD. O sistema identifica a presença de água e mostra no visor a leitura em tempo real.

Função: Exibir os dados de detecção e acionar o motor do varal.

Preço: R$ 149,90

Kit Completo com Arduino Uno

Este esquema completo utiliza o Arduino Uno, um módulo relé 5V, e o sensor FC-37. O relé comanda o motor do varal, recolhendo automaticamente quando começa a chover.

Função: Controlar todo o sistema automático de recolhimento.

Preço: R$ 199,90
